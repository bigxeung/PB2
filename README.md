# Netflix Clone React

Netflix Clone using React, TypeScript, and TMDB API

## Tech Stack

- React 18
- TypeScript
- Vite
- React Router DOM
- TMDB API
- CSS Modules / Styled Components

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Branch Strategy (Gitflow)

## Commit Convention

| Type | Description |
|------|-------------|
| feat | 새로운 기능 추가 |
| fix | 버그 수정 |
| docs | 문서 수정 |
| style | 코드 포맷팅, 세미콜론 누락 |
| refactor | 코드 리팩토링 |
| test | 테스트 코드 |
| chore | 빌드 업무, 패키지 관리 |
