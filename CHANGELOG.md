# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-09

### Added
- Initial release of Netflix Clone React
- User authentication system (login/register with TMDB API key)
- Home page with hero banner and 4 movie sections (Popular, Now Playing, Top Rated, Upcoming)
- Popular page with infinite scroll and table view toggle
- Search page with filters (genre, rating, sort) and search history
- Wishlist page with local storage persistence
- Header with scroll-based opacity animation
- Card flip animation on SignIn page
- Terms agreement checkbox for registration
- Remember Me functionality
- LocalStorage caching system (12 keys)
- Responsive design for all screen sizes
- Page transition animations
- GitHub Actions CI/CD for automatic deployment

### Technical Stack
- React 19 with TypeScript
- Vite 7 build tool
- Tailwind CSS 4
- React Router DOM v7
- Axios for API calls
- Font Awesome icons
- React Hot Toast notifications
